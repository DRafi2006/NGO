# NGO Website

This project contains everything you need to run your NGO web app locally. It is built using React, Vite, Tailwind CSS, and Shadcn UI.

## Getting Started

**Prerequisites:**

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Run the app: `npm run dev`

Open your web browser and visit `http://localhost:5173` to see the application running.
